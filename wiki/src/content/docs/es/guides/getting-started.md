## Instalación

1. Instale **NeoForge 26.1.x** (compatible con MC 26.1).
2. Coloque `mitemc-26.1.x-0.1.0.jar` en su carpeta `mods/`.
3. Inicie el juego. El mod registra `MITEMC bootstrap — Minecraft Is Too Easy, again.` si carga correctamente.

:::note
MITEMC requiere Java 21. Los lanzadores modernos de Minecraft incluyen el runtime correcto automáticamente.
:::

## Sus primeros diez minutos

1. **Aparece con 3 corazones y 3 iconos de hambre.** No corra riesgos. Una caída de 4 bloques le mata.
2. **No puede romper madera con el puño para hacer un pico.** Busque **palos bajo los árboles** o golpee hojas: cada hoja tiene posibilidad de soltar uno.
3. Con 1 palo + 1 esquirla de pedernal puede tallar una **hachuela de pedernal** en la mesa de crafteo. La hachuela abre el acceso a los troncos.
4. Coma todo lo que encuentre — manzanas de las hojas, bayas, carne cruda. El hambre disminuye incluso parado (metabolismo basal).
5. No se quede bajo la lluvia. Acelera el hambre.
6. Cuando el hambre baja de 1 icono, queda **demasiado débil para romper o colocar bloques** — lleve siempre algo de comer.
