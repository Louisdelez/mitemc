## Installation

1. Installer **NeoForge 26.1.x** (compatible MC 26.1).
2. Déposer `mitemc-26.1.x-0.1.0.jar` dans votre dossier `mods/`.
3. Lancer le jeu. Le mod écrit `MITEMC bootstrap — Minecraft Is Too Easy, again.` dans le journal s'il s'est chargé correctement.

:::note
MITEMC nécessite Java 21. Les launchers Minecraft modernes embarquent automatiquement le bon runtime.
:::

## Vos dix premières minutes

1. **Vous apparaissez avec 3 cœurs et 3 icônes de faim.** Ne prenez aucun risque. Une chute de 4 blocs vous tue.
2. **Impossible de casser du bois à mains nues pour fabriquer une pioche.** Cherchez des **bâtons sous les arbres** ou tapez les feuilles : il y a une chance d'en faire tomber un.
3. Avec 1 bâton + 1 éclat de silex, vous pouvez tailler une **hachette en silex** dans la grille de craft. La hachette ouvre l'accès aux bûches.
4. Mangez tout ce que vous trouvez — pommes des feuilles, baies, viande crue. La faim diminue même à l'arrêt (métabolisme basal).
5. Ne restez pas sous la pluie. Elle accélère la faim.
6. Quand la faim passe sous 1 icône, vous devenez **trop faible pour casser ou poser des blocs**, gardez toujours un en-cas.
