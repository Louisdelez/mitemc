# Credits & Acknowledgements

## Original MITE

**Avernite** designed and authored the original *Minecraft Is Too Easy* mod between 2014 and 2017, finishing on R196. Every survival mechanic, ore tier, tool category, and enchantment in this project traces back to that work. MITEMC is a port — not a clone — and exists only because Avernite did the hard creative work first.
